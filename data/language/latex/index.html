LaTex

Installation of latex and minted on Ubuntu

  • LaTex Installation:
 sudo apt install texlive texlive-latex-extra texlive-lang-french texmaker
  • Minted dependencies:
 sudo apt install  python3-pygments
  • In TexMaker, go in Options > Configurer Texmaker and add the following line in PdfLaTex:
 pdflatex -shell-escape -synctex=1 -interaction=nonstopmode %.tex
  • Minted example:
  \begin{minted}{python}
  import numpy as np

  def incmatrix(genl1,genl2):
      m = len(genl1)
      n = len(genl2)
      M = None #to become the incidence matrix
      VT = np.zeros((n*m,1), int)  #dummy variable
      return M
  \end{minted}

Installation of the Metropolis theme

  • Install dependencies:
  sudo apt install latexmk texlive-xetex
  • Install Metropolis with Git:

Link for the git repo: here

  cd /tmp
  git clone git@github.com:matze/mtheme.git
  cd mtheme/
  make sty && make install

Then you just need to change your latex with \usetheme{metropolis}. To use minted with Beamer, remember to add the [fragile] keyword:

  \begin{frame}[fragile]{Frame}
    \begin{minted}{c}
    int main() {
      printf("hello, world");
      return 0;
    }
    \end{minted}
  \end{frame}
